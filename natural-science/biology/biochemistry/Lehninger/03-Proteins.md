## 1 Proteins

- proteins
- 20 amino acids
- Functions of proteins
  - **catalysis** (催化) -> **hexokinase** (己糖激酶) -> DNA **polymerase** (聚合酶)
  - transport -> **hemoglobin** (血红蛋白) -> **lactose permease** (乳糖通透酶)
  - structure -> **collagen** (胶原蛋白) -> **keratin** (角蛋白)
  - motion -> **myosin** (肌球蛋白) -> **actin** (肌动蛋白)

## 2 Amino Acids

- $\alpha$-amino acids
- Amino acids = amino group + carboxyl group + side chain
- start from $\alpha$-carbon and go down the R-group
- **stereoisomerism** (立体异构现象)

- Classified into **5** main groups:

  1. Nonpolar, **aliphatic** (脂肪族) 7:

  - glycine (Gly,G, 甘氨酸) -> H- `NCC(=O)O`
  - alanine (Ala,A, 丙氨酸) -> C- `CC(N)C(=O)O`
  - proline (Pro,P, 脯氨酸) -> ring: (C1) CCC (-NC1) `OC(=O)C1NCCC1`
  - valine (Val,V, 缬氨酸) -> CC(C)- `CC(C)CNC(=O)O`
  - leucine (Leu,L, 亮氨酸) -> CC(C)C- `CC(C)CCNC(=O)O`
  - isoleucine (Ile,I, 异亮氨酸) -> CCC(C)- `CCC(C)CNC(=O)O`
  - methionine (Met, M, 甲硫氨酸, 蛋氨酸) -> CSCC- `CSCCCNC(=O)O`

  2. **Aromatic** (芳香的) R groups 3:

  - phenylalanine (Phe,F, 泵丙氨酸) -> PhC- `c1ccccc1CCNC(=O)O`
  - tyrosine (Tyr,Y, 酪氨酸) -> OPhC- `Oc1ccc(CCNC(=O)O)cc1`
  - tryptophan (Thp,W, 色氨酸) -> PhIndC- `c1cccc2c1nc(CCN(=O)O)c2`

  3. **Positively charged** R groups 3:

  - lysine (Lys,K, 赖氨酸) -> [NH3+]CCCC- `[NH3+]CCCCCNC(=O)O`
  - arginine (Arg,R, 精氨酸) -> NC(=[NH2+])NCCC- `NC(=[NH2+])NCCCCNC(=O)O`
  - histidine (His,H, 组氨酸) -> c1cnc[nH+]1C- `c1(CNC(=O)O)cnc[nH+]1`

  4. **Negatively charged** R groups 2:

  - Aspartate (Asp,D, 天冬氨酸) -> [O-]C(=O)C- `[O-]C(=O)CCNC(=O)O`
  - Glutamate (Glu,E, 谷氨酸) -> [O-]C(=O)CC- `[O-]C(=O)CCCNC(=O)O`

  5. **Polar uncharged** R groups

  - Serine (Ser,S, 丝氨酸) -> OC- `OCCNC(=O)O`
  - Threonine (Thr,T, 苏氨酸) -> CC(O)- `CC(O)CNC(=O)O`
  - Cysteine (Cys,C, 半胱氨酸)
